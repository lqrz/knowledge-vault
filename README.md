# LLM Research Vault

An Obsidian vault for learning and maintaining a durable knowledge base about large language models.

- [[topics/statistics/wiki/index|Statistics]]

Workflow:

1. Put source material in a topic's `raw/` folder.
2. Ask an LLM agent to ingest it according to `AGENTS.md`.
3. Review the generated wiki pages in Obsidian.
4. Save useful answers and study plans back into the wiki.
